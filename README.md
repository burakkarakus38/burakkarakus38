# Ahmet Burak Karakuş

Hatay Mustafa Kemal Üniversitesi Yönetim Bilişim Sistemleri (MIS) bölümü öğrencisiyim. Çalışmalarımı temel olarak backend geliştirme, veritabanı mimarisi ve siber güvenlik alanlarında yoğunlaştırmaktayım.

## Teknik Odak Noktaları

* **Veritabanı Yönetimi:** MySQL üzerinde ilişkisel veritabanı tasarımı ve yönetimi.
* **Sistem Programlama:** C ve Python dilleri ile algoritma geliştirme ve sistem mantığı çalışmaları.
* **Siber Güvenlik:** Kali Linux ortamında güvenlik araçlarının kullanımı ve test süreçleri.
* **Sistem Mimarisi:** Güvenli ve ölçeklenebilir backend sistemlerin kurgulanması.
